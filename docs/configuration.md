# Configuration

| **name**             | **default** |
|----------------------|-------------|
| title                | `"bash"`    |
| prompt_in_multiline  | `false`     |
| prompt_literal_start | `["$ "]`    |

```yaml
markdown_extensions:
  - termynal:
      title: bash
      prompt_in_multiline: false
      prompt_literal_start:
        - "$ "
```
